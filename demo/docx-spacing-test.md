# DOCX Spacing Test Suite

> 用于测试 DOCX 导出时段落、表格、引用块、代码块之间的间距补偿是否正确。
> 重点关注：行高留白偏移导致的上松下紧、表格内外文字贴边等问题。

---

## 一、基础段落间距

这是第一段普通文本。用于测试连续段落之间的间距是否均匀。段落间距应该由主题的 `spacingBefore` 和 `spacingAfter` 控制，并且经过 line-height 补偿后上下视觉等高。

这是第二段普通文本。如果补偿正确，上一段与这一段之间的视觉间距应该等于 `上一段after + 这一段before`，且段落内部文字不偏上也不偏下。

这是第三段普通文本，包含**粗体**、*斜体*和`行内代码`等内联元素，验证内联样式不影响段落整体的行盒补偿。
