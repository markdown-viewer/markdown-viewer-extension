# 表格合并功能测试 / Table Merge Demo

此文档用于测试空单元格自动向上合并功能。

## 1. 基础合并测试

| 姓名 | 部门   | 职位   |
|------|--------|--------|
| 张三 | 技术部 | 工程师 |
|      |        | 架构师 |
| 李四 | 市场部 | 经理   |

**预期效果：**
- "张三" 向下合并 2 行
- "技术部" 向下合并 2 行
- "工程师"、"架构师"、"李四"、"市场部"、"经理" 各自独立

---

## 2. 连续空单元格测试

| A | B | C |
|---|---|---|
| 1 | 2 | 3 |
|   |   | 4 |
|   |   | 5 |
| 6 | 7 | 8 |

**预期效果：**
- A 列：1 合并 3 行，6 独立
- B 列：2 合并 3 行，7 独立
- C 列：3、4、5、8 各自独立

---

## 3. 单列空测试

| 项目 | Q1 | Q2 | Q3 | Q4 |
|------|----|----|----|----|
| A    | 10 | 20 |    |    |
|      |    | 30 | 40 |    |
| B    | 50 |    |    | 60 |

**预期效果：**
- "项目" 列：A 合并 2 行，B 独立
- Q1 列：10 合并 2 行，50 独立
- Q2 列：20 独立，30 合并 2 行
- Q3 列：空 → 合并到上方（但上方是表头，不合并），40 合并 2 行
- Q4 列：两个空格合并到上方表头？不应该，因为表头不参与合并

---

## 4. 全空数据行测试

| Name | Value |
|------|-------|
| A    | 100   |
|      |       |
|      |       |
| B    | 200   |

**预期效果：**
- Name 列：A 合并 3 行，B 独立
- Value 列：100 合并 3 行，200 独立
