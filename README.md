## 📱 安装使用

### 简单 3 步开始使用

#### 第 1 步：安装扩展
1. 在 Chrome Web Store 搜索 "Markdown Viewer"
2. 点击"添加至 Chrome"按钮
3. 确认安装权限

#### 第 2 步：启用文件访问（可选）
1. 访问 `chrome://extensions/`
2. 找到 Markdown Viewer 扩展
3. 开启"允许访问文件网址"
4. 现在可以打开本地 `.md` 文件了

#### 第 3 步：开始使用
- **在线文件**：直接在浏览器中打开任何 `.md` 链接
- **本地文件**：拖拽 `.md` 文件到浏览器窗口
- **GitHub 查看**：访问任何 GitHub README 或文档文件

## ❓ 常见问题

### Q: 为什么有些本地文件打不开？
A: 需要在扩展设置中开启"允许访问文件网址"权限。

### Q: 图表为什么显示不出来？
A: 首次加载需要下载渲染库，请耐心等待几秒钟。

### Q: 如何复制图表为图片？
A: 在图表上右键选择"复制图片"，或者等待自动转换完成。

### Q: 缓存占用空间太大怎么办？
A: 点击扩展图标，在弹出窗口中点击"清理缓存"按钮。
